# 📝 Task Tracker App

A simple and interactive Task Tracker built using **HTML, CSS, and JavaScript (DOM manipulation)**.

## 🚀 Live Demo
👉 https://task-tracker-lsihoppbd-lavya-agrawal-s-projects.vercel.app

---

## ✨ Features

- ✅ Add new tasks
- ☑️ Mark tasks as complete/incomplete
- ❌ Delete individual tasks
- 🧹 Clear all completed tasks
- 📊 Task counter (Total & Completed)
- 🎨 Clean and responsive UI

---

## 🛠 Tech Stack

- HTML
- CSS
- JavaScript (DOM)
